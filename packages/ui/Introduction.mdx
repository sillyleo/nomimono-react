# Nomimono Workshop

This is a component playground for [Nomimono](https://www.npmjs.com/package/nomimono).

# Tailwind version

Zero run time and works with Next.js

```bash
pnpm install nomimono
pnpm install @fontsource/inter
```

```jsx
import {TailwindButton} from "nomimono"
import "nomimono/style"
import "@fontsource/inter/400.css";
import "@fontsource/inter/500.css";
import "@fontsource/inter/700.css";

<TailwindButton>Button</TailwindButton>
```
