# Develop components

Available scripts:


```bash
# general
pnpm build
pnpm dev
# publish package
pnpm ship
# manually build token file from Figma
# only when github workflow failed
pnpm build-token
```

## Run the dev server

```bash
pnpm dev
```

- Add files in `packages/ui/*`
- Export them in the `packages/ui/src/index.tsx` file
- Test them in `web` or `docs`

## Generate types for documentation

Generate button spec file and output it to `apps/docs/specs/button.json`

```bash filename="Run from root"
pnpm react-docgen packages/ui/Button/index.tsx -o apps/docs/specs/button.json --pretty
```

Then, in `apps/docs/pages/button.mdx`:

```jsx filename="apps/docs/pages/button.mdx"
import buttonSpec from '../specs/button.json'
import Props from '../specs/props'

<Props spec={buttonSpec} />
```
